# Movement Items

Movement items are inventory tools that primarily change how you travel around
the map.

<div class="grid cards" markdown>

- ![Hook](../assets/items/Hook.png){ width=64 } **Hook** — reward for completing [Level 1](../world/locations/level-1.md). Allows acrobatic grappling around the map.
- ![Golden Hook](../assets/items/Golden_Hook.png){ width=64 } **Golden Hook** — reward for completing the [Log Challenge](../world/locations/log-challenge.md). Same function as the Hook but gold-colored.
- ![Insta-Horse](../assets/items/Insta-Horse.png){ width=64 } **Insta-Horse** — sold in the [Port Town Shop](../world/locations/port-town.md#shop). Spawns a Horse, but can only be used outside of safe zones.

</div>
