# Movement Items

Movement items are inventory tools that primarily change how you travel around
the map.

<div class="grid cards" markdown>

- ![Hook](../assets/items/Hook.png){ width=64 } **Hook** — reward for completing [Level 1](../world/locations/level-1.md). Allows acrobatic grappling around the map, but lasts only about **15 seconds** after being fired. Its shot drops off faster and does not reach as far as the Golden Hook.
- ![Golden Hook](../assets/items/Golden_Hook.png){ width=64 } **Golden Hook** — reward for completing the [Log Challenge](../world/locations/log-challenge.md). A stronger version of the Hook with longer range and a much longer duration, lasting about **1 minute 2 seconds** after being fired.
- ![Insta-Horse](../assets/items/Insta-Horse.png){ width=64 } **Insta-Horse** — sold in the [Port Town Shop](../world/locations/port-town.md#shop). Spawns a Horse, but can only be used outside of safe zones.

</div>
